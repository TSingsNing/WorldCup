# 微信小程序注册与运行说明

作者：**Manus AI**

## 一、你需要注册和准备的内容

如果只是本地体验本仓库代码，你可以先不注册正式小程序账号，直接用微信开发者工具的测试号或临时 AppID 导入项目运行。如果希望发布到微信线上环境，则需要在微信公众平台完成小程序注册、主体信息填写、开发者绑定、服务类目选择、代码上传、审核和发布。微信官方文档说明，小程序账号需要在微信公众平台完成注册，并在“设置-开发设置”中获取 AppID；完成开发后，还需要在后台提交审核，审核通过后手动发布。[1]

| 场景 | 是否必须注册小程序账号 | 你需要做什么 |
| --- | --- | --- |
| 本地开发和预览 | 否 | 安装微信开发者工具，导入仓库，使用测试号或替换为已有 AppID。 |
| 真机体验测试 | 建议需要 | 注册小程序账号，获取 AppID，并在后台绑定项目成员/体验成员。 |
| 正式上线发布 | 必须 | 完成主体信息、小程序信息、类目配置、代码上传、审核和发布。 |
| 使用微信支付 | 需要认证主体 | 官方文档说明，已认证小程序可申请微信支付；个人主体暂不支持微信认证。[1] |

## 二、注册流程概览

你需要访问[微信公众平台](https://mp.weixin.qq.com/)，点击“立即注册”，账号类型选择“小程序”。随后填写一个未注册过公众平台、开放平台、企业号且未绑定个人微信号的邮箱，激活邮箱后继续选择主体类型，并填写主体信息和管理员信息。微信官方列出的主体类型包括个人、企业、个体工商户、政府、媒体和其他组织。[1]

> 微信官方文档说明：“完成注册后，微信小程序信息完善步骤和开发可同步进行。”这意味着你可以在完善名称、头像、介绍和服务范围的同时推进代码开发。[1]

| 主体类型 | 适用情况 | 关键限制或说明 |
| --- | --- | --- |
| 个人 | 个人兴趣、学习、轻量工具 | 需为 18 岁以上且具备国内身份信息的微信实名用户；个人主体暂不支持微信认证。 |
| 企业/个体工商户 | 商业化产品、品牌运营 | 可选择对公账户打款验证或微信认证；如需微信支付通常需要认证。 |
| 政府、媒体、其他组织 | 机构类应用 | 必须通过微信认证，认证前部分能力不可用。 |

## 三、获取 AppID 与绑定开发者

注册完成后，你需要登录小程序后台，在“设置-开发设置”中获取 AppID。开发前还需要进入“用户身份-成员管理”添加项目成员。官方文档说明，个人主体和非个人主体在不同状态下可绑定的项目成员与体验成员数量不同；本项目开发阶段只需要把实际开发者微信号加入项目成员即可。[1]

拿到 AppID 后，可以将仓库根目录的 `project.config.json` 中 `appid` 字段从 `touristappid` 替换为你的真实 AppID。若你暂时没有 AppID，可以保持默认值，并在微信开发者工具中选择测试号模式导入。

## 四、本地运行步骤

| 步骤 | 操作 |
| --- | --- |
| 1 | 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/devtools.html)。官方工具支持小程序模拟器、调试器、代码查看、预览和发布等能力。[2] |
| 2 | 打开微信开发者工具，选择“导入项目”。 |
| 3 | 项目目录选择本仓库根目录 `WorldCup`。 |
| 4 | AppID 可使用测试号，也可填入你的真实小程序 AppID。 |
| 5 | 点击“编译”，进入首页后选择难度即可开始答题；首页也可以手动开启背景音乐。 |
| 6 | 全部答对后进入证书页，点击“保存后发群炫耀”时，模拟器可能需要授权相册权限；真机上也需要用户允许保存到相册。 |
| 7 | 点击“好友段位榜”可以查看本地排行榜原型：本机通关成绩会写入缓存，其他好友数据为模拟占位。 |

## 五、代码上传与发布

当你准备发布时，需要在微信开发者工具中点击“上传”生成开发版本。随后登录微信公众平台小程序后台，在“版本管理”中找到开发版本并提交审核。审核时需要配置重要业务页面的类目与标签；如果小程序需要登录账号才能体验，还需要按官方流程提供测试账号。审核通过后，管理员仍需手动点击发布，线上用户才能访问。[1]

## 六、当前版本注意事项

当前版本是纯前端小程序 MVP，已包含本地排行榜原型，但尚未接入真实微信登录、真实好友关系、后端题库、内容安全接口、AI 在线出题和云存储。题目集中在 `utils/questions.js`，可以直接修改后重新编译。证书图片由 `pages/certificate/certificate.js` 使用 Canvas 绘制，后续可以扩展昵称输入、头像、分享海报、小程序码或更精细的视觉模板。

背景音乐位于 `assets/audio/worldcup_quiz_bgm.mp3`，由 `app.js` 中的全局 App 实例统一控制。微信小程序音频播放通常需要用户手势触发，因此建议保留首页 BGM 开关，不建议完全依赖自动播放。

如果要上线真实好友排行榜，建议接入微信云开发或自有后端，将用户的 `openid`、昵称、头像、分数、难度、段位和用时写入数据库。需要注意，微信小程序不能无条件读取用户完整好友列表；真实好友榜应在用户授权和平台能力边界内设计。
