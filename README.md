# WorldCup 微信小程序

作者：**Manus AI**

## 项目简介

本仓库实现了一个**世界杯答题互动微信小程序**原型。用户进入小程序后可以选择不同难度的世界杯知识挑战，连续答对全部题目后即可生成一张专属荣誉证书图片。第一版采用原生微信小程序开发，不依赖后端服务，题库和证书生成逻辑均在端上完成，因此可以直接通过微信开发者工具导入并运行。

## 核心功能

| 功能 | 当前实现 |
| --- | --- |
| 难度选择 | 已实现“球迷入门”“战术观察家”“传奇考古家”三档难度。 |
| 世界杯题库 | 已内置 18 道题，覆盖基础常识、经典决赛、历史纪录和冷门知识。 |
| 答题互动 | 已实现单选答题、即时正误反馈、答案解析、连对进度和失败重来。 |
| 证书生成 | 已实现全对后进入证书页，通过 Canvas 绘制荣誉证书。 |
| 保存图片 | 已实现调用微信相册能力保存证书图片。 |
| 本地运行 | 已提供 `project.config.json`，可用微信开发者工具直接导入。 |

## 目录结构

| 路径 | 说明 |
| --- | --- |
| `app.js`、`app.json`、`app.wxss` | 小程序全局入口、页面路由和全局样式。 |
| `pages/index` | 首页和难度选择页。 |
| `pages/quiz` | 答题页，包含题目加载、选择判断、反馈与跳转逻辑。 |
| `pages/certificate` | 证书页，包含 Canvas 绘制和保存相册逻辑。 |
| `utils/questions.js` | 本地世界杯题库和难度元数据。 |
| `docs/architecture.md` | 页面结构和交互流程设计。 |
| `docs/registration-and-run.md` | 微信小程序注册、AppID 获取、开发者工具运行和发布说明。 |

## 本地运行方式

请先安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/devtools.html)，然后选择“导入项目”，项目目录选择本仓库根目录。如果只是本地开发或预览，可以使用测试号或将 `project.config.json` 中的 `appid` 替换为你自己的小程序 AppID。导入后点击“编译”即可运行。

## 后续可扩展方向

当前版本是可运行的 MVP。后续可以继续扩展微信登录、排行榜、分享海报、题库运营后台、AI 动态出题、错题本、赛事专题题包和用户成长体系。如果需要多人数据和长期运营，建议接入微信云开发或自有后端服务。
