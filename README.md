# WorldCup 微信小程序：世界杯预言家

## 项目简介

本仓库当前实现了一个面向世界杯球迷的**合规娱乐预测与好友 PK 微信小程序原型**。产品定位为“世界杯预言家”：用户可以查看热点赛程，进行每日赛果、比分、进球数、首球时间、冠军和金靴等娱乐预测，并通过热力积分、好友房间排行榜和个人身份卡获得社交传播动机。

当前版本采用**原生微信小程序**开发，不依赖后端服务，赛程数据、预测记录、房间信息、身份卡画像和模拟排行榜均在端上完成，因此可以直接用微信开发者工具导入运行。正式上线后，如果需要真实用户体系、真实好友房间、跨设备同步和运营后台，建议接入微信云开发或自有后端服务。

## 核心功能

| 功能 | 当前实现 |
| --- | --- |
| 单页聚合首页 | 首页集中展示产品定位、预测状态、热点赛程、好友房间、身份卡、AI 分析入口和合规提示，减少用户来回翻页。 |
| 每日娱乐预测 | 支持选择热点赛程，预测胜平负/爆冷、比分、进球区间、首球时间、冠军和金靴，并保存到本地缓存。 |
| AI 赛前分析 | 内置静态分析文案，展示走势、看点、娱乐比分和爆冷观察，强调不构成投注或收益建议。 |
| 好友房间 PK | 支持切换“天台杯”“熬夜看球群”等房间，展示模拟好友排行榜与本机预测积分。 |
| 身份卡传播 | 用户可配置昵称、支持球队、冠军预测和金靴预测，并生成“黄金预言家/反向明灯”等身份卡。 |
| Canvas 海报 | 身份卡页使用 Canvas 绘制分享海报，并调用微信相册能力保存图片。 |
| 背景音乐 | 复用现有世界杯 BGM 管理能力，首页和核心页面可尝试播放或切换 BGM。 |
| 合规边界 | 页面明确声明不涉及充值、提现、现金奖品、真实投注、赔率、盘口或可转让虚拟币。 |
| 分享能力 | 首页、预测页、身份卡页、排行榜页均保留 `onShareAppMessage` 与 `onShareTimeline`。 |

## 目录结构

| 路径 | 说明 |
| --- | --- |
| `app.js`、`app.json`、`app.wxss` | 小程序全局入口、页面路由、窗口配置和全局样式。 |
| `assets/audio/*.m4a` | 世界杯主题 BGM 素材，继续复用原项目音频资源。 |
| `pages/index` | 首页，聚合产品介绍、预测入口、赛程、个人状态、好友房间、身份卡和合规提示。 |
| `pages/quiz` | 娱乐预测页，包含赛程切换、AI 赛前分析、预测表单和本地保存逻辑。 |
| `pages/certificate` | 身份卡页，包含球迷画像配置、Canvas 海报绘制、保存相册和分享文案。 |
| `pages/leaderboard` | 好友房间 PK 页，展示房间切换、本地预测积分、模拟好友榜和积分规则。 |
| `utils/worldcup.js` | 世界杯娱乐预测数据层，包含赛程、热度、合规文案、本地存储、画像和排行榜聚合。 |
| `utils/questions.js`、`utils/rank.js` | 原答题版本遗留工具，当前主流程不再依赖，可作为后续世界杯知识挑战玩法扩展素材。 |

## 本地运行方式

请先安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/devtools.html)，然后选择“导入项目”，项目目录选择本仓库根目录。如果只是本地开发或预览，可以使用测试号，或将 `project.config.json` 中的 `appid` 替换为你自己的小程序 AppID。导入后点击“编译”即可运行。

需要注意的是，微信小程序的音频自动播放能力会受到用户手势和系统权限限制。因此当前版本保留了显式 BGM 开关，并在用户进入核心页面后尝试播放背景音乐，这更符合小程序实际运行环境。

## 合规设计边界

本项目按“娱乐预测 + 社交荣誉”的方向设计，不实现真实竞猜、博彩、投注或现金回报机制。页面文案刻意避免“下注、买球、庄家、返奖、赔率、盘口、水位、梭哈”等表达，积分仅用于荣誉展示与好友排名，不可充值、不可提现、不可转让，也不兑换现金或高价值实物。

> 产品底层机制应保持为**预测、排行榜、徽章、身份卡和社交传播**，而不是博彩交易。若后续上线运营，建议在用户协议、活动规则、审核资料和页面显著位置持续保留该边界说明。

## 后续可扩展方向

当前版本已经具备可运行 MVP 的主体闭环。后续可以接入微信云开发实现真实登录、真实房间、云端赛程、赛果结算、运营后台、好友邀请码、用户徽章体系和动态赛前分析。如果需要 AI 动态生成赛前分析，建议由后端统一调用模型并进行内容审核，避免端上暴露密钥或生成不合规文案。
