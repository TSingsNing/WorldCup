# WorldCup 微信小程序

作者：**Manus AI**

## 项目简介

本仓库实现了一个**世界杯像素答题互动微信小程序**原型。用户进入小程序后可以选择不同难度的世界杯知识挑战，系统会从题池中随机抽题并打乱选项，连续答对全部题目后即可生成一张带有“球格人格”、段位、分数和通关称号的专属证书图片。第二版在第一版可运行 MVP 基础上进一步强化了娱乐化氛围、背景音乐、排行榜和分享欲设计。

当前版本采用**原生微信小程序**开发，不依赖后端服务，题库、排行榜原型和证书生成逻辑均在端上完成，因此可以直接通过微信开发者工具导入并运行。正式上线后，如果需要真实好友排行榜和跨设备数据同步，建议接入微信云开发或自有后端服务。

## 核心功能

| 功能 | 当前实现 |
| --- | --- |
| 难度选择 | 已实现“球迷入门局”“战术观察局”“传奇考古局”“球史怪物局”四档难度。 |
| 世界杯题库 | 已内置 38 道题，覆盖基础常识、经典决赛、历史纪录、冷门赛制和资深球迷挑战题。 |
| 随机抽题 | 每局从当前难度题池随机抽取指定数量题目，并对选项做洗牌，降低重复感。 |
| 背景音乐 | 已加入原创世界杯答题 BGM，首页可手动开启/暂停，开始答题后自动尝试播放。 |
| 答题互动 | 已实现单选答题、即时正误反馈、答案解析、连对进度、震动反馈和失败重开。 |
| 个性化证书 | 全对后通过 Canvas 绘制证书，包含球格人格、段位、用时、分数和趣味文案。 |
| 保存图片 | 已实现调用微信相册能力保存证书图片，便于分享到聊天或朋友圈。 |
| 好友段位榜 | 已实现本地排行榜原型：本机通关成绩真实写入缓存，好友数据用模拟选手占位。 |
| 本地运行 | 已提供 `project.config.json`，可用微信开发者工具直接导入。 |

| `app.js`、`app.json`、`app.wxss` | 小程序全局入口、页面路由和全局样式。 |
| `assets/audio/worldcup_quiz_bgm.mp3` | 原创无歌词背景音乐。 |

## 本地运行方式

请先安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/devtools.html)，然后选择“导入项目”，项目目录选择本仓库根目录。如果只是本地开发或预览，可以使用测试号或将 `project.config.json` 中的 `appid` 替换为你自己的小程序 AppID。导入后点击“编译”即可运行。

需要注意的是，微信小程序的音频自动播放能力会受到用户手势和系统权限限制。因此第二版在首页提供了显式 BGM 开关，并在用户点击开始答题后尝试播放背景音乐，这更符合小程序实际运行环境。

## 真实好友排行榜上线建议

当前排行榜为了保证本地可运行，采用“本机缓存成绩 + 模拟好友数据”的方式实现。若要上线真实好友段位榜，可以优先选择微信云开发：使用 `openid` 作为用户身份标识，将每次通关记录写入云数据库，并按分数、难度权重和用时排序。若需要微信好友关系能力，则还需要结合小程序开放能力与用户授权边界设计，不能直接任意读取用户微信好友列表。

## 后续可扩展方向

当前版本已经具备可玩 MVP 的主体闭环。后续可以继续扩展微信登录、真实排行榜、分享海报、题库运营后台、AI 动态出题、错题本、赛事专题题包、每日挑战、限时模式和用户成长体系。如果需要多人数据和长期运营，建议接入微信云开发或自有后端服务。
